## 1. LiDAR SLAM / Sensor Fusion

### 신규 논문 (arXiv)

- **[LIT-GS: LiDAR-Inertial-Thermal Gaussian Splatting for Illumination-Robust Mapping](https://arxiv.org/abs/2606.20424)** — 미상 저자. arXiv:2606.20424 (2026-06-20 경)
  - LiDAR로 추출한 평면 기하를 명시적 구속 조건으로 삽입해 Gaussian 최적화와 포즈 추정 양쪽에 적용. 열화상 센서를 추가로 융합해 조도 변화·텍스처 부족 환경에서의 강건성을 대폭 향상. 기존 LiDAR-Inertial-Visual 파이프라인이 조명에 취약한 근본 원인을 해결한 점에서 실내외 야간 매핑 응용에 직접적으로 의미 있음.

- **[SGM-SLAM: Scene Graph Matching for Data-Efficient Distributed SLAM](https://arxiv.org/abs/2606.16881)** — Yewei Huang, Tixiao Shan, Abhinav Rajvanshi et al. arXiv:2606.16881, IEEE RAL 2026 accepted
  - LiDAR + 카메라 + IMU를 탑재한 다중 로봇 팀을 위한 분산 SLAM 프레임워크. 통신 대역을 최소화하기 위해 객체 레이블·centroid만으로 씬 그래프 매칭을 수행하는 최초 접근. GPS 없이 대규모 실내외 환경의 장기 임무에서 로봇 간 측정 제약을 효율적으로 공유.

- **[Semantic-weighted ICP for LiDAR Odometry: Class-Aware Residual Reweighting for Robust Scan Registration](https://arxiv.org/abs/2606.03905)** — Vasco Carvalho, Tiago Barros, Urbano J. Nunes. arXiv:2606.03905 (2026-06-02)
  - 특정 의미 클래스(동적 객체 등)를 단순 필터링하는 대신 클래스별 기하 안정성에 따라 ICP 잔차를 재가중치. SemanticKITTI 및 RELLIS-3D 검증. 동적 환경 주행 오도메트리 정확도 개선에 실용적 기여.

- **[RadiusFPS: Efficient Farthest Point Sampling on CPUs and GPUs via Spherical Voxel Pruning](https://arxiv.org/abs/2606.06255)** — arXiv:2606.06255 (2026-06-06)
  - 포인트 클라우드 다운샘플링의 병목인 FPS(Farthest Point Sampling)를 구형 복셀 프루닝으로 가속. CPU·GPU 양쪽에서 속도 향상. LiDAR SLAM 전처리 및 대규모 3D 인식 파이프라인에서 즉시 적용 가능.

## 2. Path planning / Motion planning

### 신규 논문 (arXiv)

- **[Accelerating and Scaling MPC-Guided Reinforcement Learning for Humanoid Locomotion and Manipulation](https://arxiv.org/abs/2606.05687)** — arXiv:2606.05687 (2026-06-05)
  - 휴머노이드 로봇의 전신 제어에서 MPC의 물리 기반 예측·제약 처리 능력과 RL의 대규모 시뮬레이션 기술을 결합. MPC가 RL 탐색을 가이드해 수렴 속도와 스케일 양쪽을 동시에 개선. 현재 유행하는 MPC+RL 융합 패러다임의 실용적 구현 사례.

- **[ConsistencyPlanner: Real-time Planning with Fast-Sampling Consistency Models](https://arxiv.org/abs/2606.11569)** — Qichao Zhang, Xing Fang, Jiaqi Fang et al. (CAS, Guangzhou Zaofu). arXiv:2606.11569 (2026-06-10)
  - Consistency Model의 빠른 샘플링 특성을 활용해 다중 모달 미래 궤적 집합을 실시간으로 생성, 폐루프 자율주행 플래닝에 적용. 기존 Diffusion 기반 플래너의 추론 속도 병목을 실질적으로 해소. 자율주행 실시간성 요건 충족에 직접 기여.

- **[Uncertainty-Aware Motion Planning for Autonomous Driving in Mixed Traffic Environment](https://arxiv.org/abs/2606.09958)** — arXiv:2606.09958 (2026-06-09)
  - AV와 인간 운전 차량이 혼재하는 환경에서 인간 의도 예측의 불확실성을 플래닝 단계까지 전파하는 UAMP 제안. 의도 예측 오차에 강건한 안전 주행 결정에 특화.

- **[Towards End-to-End Motion Planning and Execution for Autonomous Underwater Vehicles Using Reinforcement Learning](https://arxiv.org/abs/2606.08513)** — arXiv:2606.08513 (2026-06-06)
  - 원시 센서 데이터→스러스터 명령 매핑을 두 단계 계층적 MDP로 분리해 학습. 수중 AUV는 GPS·통신 불가 환경의 극단적 사례이므로 강건 엔드투엔드 플래닝 연구 방향성을 보여주는 선행 논문.

- **[Increasing Resilience of Continuum Robots via Motion Planning Algorithms](https://arxiv.org/abs/2606.20495)** — arXiv:2606.20495 (2026-06-18)
  - 연속체 로봇(continuum robot)을 위한 경로 계획에 다목적 의사결정(AHP+유전 알고리즘, A*)을 결합해 경로 품질 평가. 의료·협소 공간 탐사 로봇 분야에서 운동 계획 다양화를 다룬 실험적 연구.

- **[Convex-Neural RRT*: Fast and Reliable Learning-Guided Sampling for High-Quality Robot Path Planning](https://arxiv.org/abs/2605.25006)** `[catch-up]` — arXiv:2605.25006 (2026-05 제출)
  - 신경망 가이드 샘플링과 볼록 집합 기반 최적화를 결합해 RRT*의 수렴 속도를 가속. 고차원 구성 공간에서 품질 높은 경로를 빠르게 획득. 지난 주 미수록 아이템으로 catch-up.

## 3. 3D Reconstruction / Scanning

### 신규 논문 (arXiv)

- **[Envision4D: Envisioning Visual Futures via Feed-forward 4D Gaussian Splatting for Autonomous Driving](https://arxiv.org/abs/2606.10656)** — (Tsinghua Univ., CUHK Shenzhen). arXiv:2606.10656 (2026-06-09)
  - 미래 카메라 포즈를 반복적 노이즈 제거 프로세스로 추론하는 Future Pose Prediction 모듈 + In-layer Temporal Attention + Conditioned Motion Lifting을 결합. 포즈 라벨 없이 동적 자율주행 장면의 미래 시점을 합성해 state-of-the-art 성능. 시뮬레이션 데이터 생성·안전 검증 시나리오 확충에 즉시 응용 가능.

- **[GASE: Gaussian Splatting-Based Automated System for Reconstructing Embodied-Simulation Environments](https://arxiv.org/abs/2606.17520)** — arXiv:2606.17520 (2026-06-20 경)
  - 파노라믹 카메라 어레이의 다시점 비디오 스트림에서 시뮬레이션 씬을 완전 자동으로 구축. 구체화 AI(Embodied AI) 학습용 대규모 환경 데이터 확보 파이프라인으로서, 수작업 씬 제작 비용 절감 효과가 큼.

- **[A Scalable PyTorch Abstraction for Multi-GPU Gaussian Splatting](https://arxiv.org/abs/2606.11390)** — arXiv:2606.11390 (2026-06-10)
  - 10억 개 이상의 Gaussian splat으로 구성된 도시 규모 재구성을 다중 GPU로 분산 처리하는 PyTorch 추상화 계층. 코드 복잡성을 감추면서 도시 규모 3DGS 재구성을 실용화하는 엔지니어링 기여.

- **[Geometry Gaussians: Decoupling Appearance and Geometry in Gaussian Splatting](https://arxiv.org/abs/2606.05124)** — arXiv:2606.05124 (2026-06-05)
  - 3DGS에서 외관(appearance)과 기하(geometry)를 명시적으로 분리해 정확한 기하 정보 추출 정확도를 높이는 방법. 3DGS를 3D 메쉬·포인트 클라우드 후처리에 활용할 때의 기하 품질 문제를 직접 해결.

- **[Optimizing 3D Gaussian Splatting via Point Cloud Upsampling](https://arxiv.org/abs/2606.00450)** — arXiv:2606.00450 (2026-06-01)
  - 3DGS 초기화 품질이 최종 재구성 품질에 미치는 영향에 주목, 선형 보간·삼각형 보간·보로노이 기반 포인트 생성 등 업샘플링 기법을 비교 분석. 희소 SfM 포인트 클라우드로부터 고품질 Gaussian 초기화 달성.

- **[A Cookbook of 3D Vision: Data, Learning Paradigms, and Application](https://arxiv.org/abs/2606.04291)** — Brown Univ., Univ. of Maryland, Univ. of Pennsylvania 등. arXiv:2606.04291 (2026-06-02)
  - 3D 비전 전반(데이터·학습 패러다임·응용)을 망라하는 교과서급 서베이. 체적 재구성·객체 모델링까지 포함. 신규 연구자 온보딩 및 분야 전체 조망에 유용.
