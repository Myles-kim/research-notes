## 1. LiDAR SLAM / Sensor fusion

### 신규 논문 (arXiv)

- **[Uncertainty-Aware Adaptive Sensor Fusion for Autonomous Navigation]** — Multiple authors. arXiv:2606.05437 (2026-06-03)
  https://arxiv.org/abs/2606.05437
  - Vision Transformer(ViT)로 IMU 시계열을 처리하고 MCNN으로 광학흐름 기반 시각 특징을 추출한 뒤, UKF와 결합한 하이브리드 VIO. 불확실성 추정 기반 적응형 센서 가중치로 도전적 환경에서 강인성 확보. LiDAR-IMU 계열 연구자에게도 적응형 퓨전 설계 인사이트 제공.

- **[D-LIO: 6DoF Direct LiDAR-Inertial Odometry based on Simultaneous Truncated Distance Field Mapping]** — Multiple authors. arXiv:2505.16726 (IEEE RA-L 2026)
  https://arxiv.org/abs/2505.16726
  [catch-up] 피처 추출 없이 원시 3D LiDAR 포인트를 Truncated Distance Field에 직접 매핑해 6DoF odometry를 CPU 실시간 처리. FAST-LIO2 이후 가장 실용적인 피처-프리 LiDAR-IMU 방법론으로 RA-L 2026 게재 확정.

- **[RU4D-SLAM: Reweighting Uncertainty in Gaussian Splatting SLAM for 4D Scene Reconstruction]** — Multiple authors. arXiv:2602.20807
  https://arxiv.org/abs/2602.20807
  [catch-up] 동적 장면의 4D 재구성(3D+시간) 문제에 불확실성 재가중치를 적용한 3DGS 기반 SLAM. GS-SLAM 계열에서 동적 객체 처리를 정면으로 다루는 초기 사례.

## 2. Path planning / Motion planning

### 신규 논문 (arXiv)

- **[Uncertainty-Aware Motion Planning for Autonomous Driving in Mixed Traffic Environment]** — Ming Cheng, Hao Chen, Ziyi Yang, Ziluowen Luo, Senzhang Wang. arXiv:2606.09958 (2026-06-08)
  https://arxiv.org/abs/2606.09958
  - 혼합 교통(AV+인간 운전 차량) 환경에서 주변 차량 의도의 내재적 불확실성을 명시적으로 정량화하는 UAMP 프레임워크. 근접도 기반 불확실성 추정기로 인터랙션 조건부 의도 분포를 구성, 불확실성 안내 조인트 분포로 장기 의사결정 개선. 결정론적 예측 의존에 따른 위험 행동을 방지하는 실용적 접근.

- **[AgenticDiffusion: Agentic Diffusion-based Path Planning for Vision-Based UAV Navigation]** — Multiple authors. arXiv:2606.04111 (2026-06)
  https://arxiv.org/abs/2606.04111
  - 실내 UAV 항법에서 언어 지시·FPV+탑뷰 멀티뷰 관측·오픈보캐뷸러리 타겟 그라운딩·뷰포인트별 확산 플래너·NMPC 실행을 단일 에이전틱 파이프라인으로 통합. "지시→계획→실행"을 일관된 멀티뷰 추론 위에 구현, 폐색·타겟 가시성 한계 극복. 자연어 제어 드론 자율성의 새로운 기준점.

- **[Sample-efficient Low-level Motion Planning for Robotic Manipulation Tasks via Zero-shot Transfer Learning]** — Yuanzhi He, Victor Romero-Cano, José J. Patiño, Juan David Hernández, William Sawtell, Gualtiero Colombo. arXiv:2606.06041 (2026-06)
  https://arxiv.org/abs/2606.06041
  - 진화 알고리즘 iCEM에 전이 학습을 결합한 iCEM+TL 프레임워크. 상위 태스크 파라미터를 하위 복잡 태스크로 제로샷 전이해 쌓기·슬라이딩·선반 배치 성능 개선. 실제 Franka Emika 로봇에서 검증. Reward Redesign으로 태스크별 최적화 병행.

- **[Towards End to End Motion Planning and Execution for Autonomous Underwater Vehicles Using Reinforcement Learning]** — Multiple authors. arXiv:2606.08513 (2026-06)
  https://arxiv.org/html/2606.08513
  - AUV(자율 수중 로봇)의 원시 센서 입력을 스러스터 명령으로 직접 매핑하는 E2E RL 플래닝. 수중 환경 특성(통신 지연·제한적 시야)에 특화된 설계로 AUV 자율 운용 시나리오를 직접 타겟.

- **[Sampling-Based Motion Planning with Scene Graphs Under Perception Constraints]** — Multiple authors. arXiv:2603.03514 (2026-03)
  https://arxiv.org/html/2603.03514v1
  [catch-up] 씬 그래프와 샘플링 기반 플래닝을 결합해 지각 제약(FOV 한계·센서 범위) 하에서 안전 궤적 생성. 인간 중심 복잡 환경의 로봇 운용 시나리오를 직접 타겟. RRT* 계열에 의미론적 구조 이해를 더한 진화 방향.

## 3. 3D Reconstruction / Scanning

### 신규 논문 (arXiv)

- **[Optimizing 3D Gaussian Splatting via Point Cloud Upsampling]** — Multiple authors. arXiv:2606.00450 (2026-05-30)
  https://arxiv.org/html/2606.00450v1
  - 3DGS 렌더링 품질이 SfM 초기 시드 포인트 밀도에 크게 의존한다는 구조적 한계를 정면 공략. 선형·삼각·스플라인·MLS·Voronoi 등 5가지 포인트클라우드 업샘플링 방법을 체계적으로 비교 평가. depth map 기반 geometric consistency를 유지하는 depth-guided point lifting 방법 추가 제안. 3DGS 실용 파이프라인 품질 향상에 즉시 적용 가능한 엔지니어링 기여.

- **[RadiusFPS: Efficient Farthest Point Sampling on CPUs and GPUs via Spherical Voxel Pruning]** — Multiple authors. arXiv:2606.06255 (2026-06-04)
  https://arxiv.org/html/2606.06255
  - 포인트클라우드 처리의 핵심 병목인 FPS(Farthest Point Sampling)를 구형 복셀 가지치기(Spherical Voxel Pruning)로 CPU/GPU 모두에서 가속. 현대 3D LiDAR의 초당 수백만 포인트 처리에 직결되는 지연 개선. SLAM 및 3D 재구성 전처리 파이프라인 전반의 실시간성 향상 기여.

- **[Faster 3D Gaussian Splatting Convergence via Structure-Aware Densification]** — Multiple authors. arXiv:2604.28016 (2026-04-30)
  https://arxiv.org/abs/2604.28016
  [catch-up] 기존 3DGS의 반복적 고밀도화 단계를 조기 구조 인식 고밀도화(early structure-aware densification)로 대체해 수렴 속도를 대폭 단축. 고주파 영역 재구성 품질도 동시 향상. 3DGS 학습 비용 절감에 가장 실질적인 최근 기여.

- **[ELoG-GS: Dual-Branch Gaussian Splatting with Luminance-Guided Enhancement for Extreme Low-light 3D Reconstruction]** — Multiple authors. arXiv:2604.12592 (2026-04-14)
  https://arxiv.org/abs/2604.12592
  [catch-up] NTIRE 2026 3D 복원 챌린지 참가작. 극저광도 멀티뷰 입력에서 듀얼 브랜치 GS + 휘도 안내 향상으로 고품질 3D 표현 복원. 야간 자율주행·야간 검사 로봇 등 실용 시나리오에 직접 적용 가능한 첫 저조도 특화 3DGS.

- **[Dy3DGS-SLAM: Monocular 3D Gaussian Splatting SLAM for Dynamic Environments]** — Mingrui Li et al. arXiv:2506.05965 (2025-06-06)
  https://arxiv.org/abs/2506.05965
  [catch-up] 모노큘러 RGB만으로 동적 장면 SLAM을 구현한 최초의 3DGS 기반 방법. 광학흐름 마스크+깊이 마스크를 확률 모델로 융합해 동적 객체 자동 제거. 스케일 복원 및 포즈 추정 동시 수행. 1년이 지난 지금도 GS-SLAM 동적 처리 연구의 기준 논문으로 인용 증가세.
